▶ Anleitung anzeigen
▶ Funktionen
1
Liedtext einfügen
Links einfügen oder Datei öffnen (TXT, Word, PDF). Abschnitte mit Leerzeile trennen. Optional mit Strophe, Refrain oder Bridge beschriften.
2
Übersetzung einfügen
In der Mitte einfügen — gleiche Struktur. Oder KI-Übersetzung nutzen (immer prüfen!).
3
Ausgabe rechts
Erscheint automatisch. Pro Folie max. 3 Zeilenpaare. Kopieren oder als .txt speichern.
?
Tipps
Zeilen aufteilen trennt abwechselnde DE/RU-Zeilen automatisch. I: Text :| = Wiederholung. -- = manueller Folientrenner.
📂
Datei-Import
TXT, Word (.docx), PDF und Bilder (JPG, PNG). Auch per Drag & Drop auf die Textfelder.
🔍
OCR-Texterkennung
Bilder und gescannte PDFs werden per Google Vision API erkannt. Akkorde, Copyright und Liednummern werden automatisch entfernt.
🎵
Audio-Import
MP3, WAV, OGG und weitere Formate. Das Lied wird per KI transkribiert (Deutsch & Russisch).
✨
KI-Bereinigung
Nach einem Import kann der Text per Gemini-KI bereinigt werden: Zeilen richtig zusammensetzen, Silben verbinden, Refrain wiederholen.
🌐
KI-Übersetzung
Automatische Übersetzung DE↔RU per Google Translate. Die Zeilenstruktur wird gespiegelt.
🔄
Struktur-Sync
Wenn aktiviert, werden Leerzeilen-Änderungen im Liedtext automatisch in die Übersetzung übernommen.
⇅
Zeilen aufteilen
Erkennt abwechselnde DE/RU-Zeilen und trennt sie automatisch in Liedtext + Übersetzung.
📄
Formatierung
Automatische Folien-Aufteilung (max. 3 Zeilenpaare), Strophen-/Refrain-Labels, |: :| Wiederholungen, -- manuelle Trenner.
💾
Export
Ausgabe als .txt speichern (Dateiname = erste Liedzeile) oder in die Zwischenablage kopieren.
🌙
Helles/Dunkles Design
Umschaltbar über den Button oben rechts. Die Wahl wird gespeichert.
